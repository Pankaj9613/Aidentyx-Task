 User Management Application
 Overview

This is a full-stack User Management Application developed using:

Backend: Spring Boot

Frontend: React

Database: PostgreSQL

The application allows users to:

Create a new user

View existing users

Import multiple users using a CSV file

Store passwords securely using encryption

Technology Stack
Backend

Java 17

Spring Boot 3

Spring Data JPA

PostgreSQL

Spring Security (for password encryption)

Frontend

React 18

Axios

Database

PostgreSQL
